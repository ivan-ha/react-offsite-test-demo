# React Demo
React demo with sticky navbar, infinite scroll and user login

### Build method
1. `npm install`
2. `npm start`
3. Visit [localhost:8080](localhost:8080)

### Reserved ports
| Port | Usage                                                                                |
|------|--------------------------------------------------------------------------------------|
| 3000 | Mock API server using [json-server](https://github.com/typicode/json-server#install) |
| 8080 | Web server                                                                           |
